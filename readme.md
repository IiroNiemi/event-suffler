# Event suffler

## Installation
Please use docker-compose to install Event suffler.

[docker-compose](https://docs.docker.com/compose/install/)

```bash
docker-compose -f docker-compose.yml build
docker-compose -f docker-compose.yml up
```

## Run local tests
```bash
npm install
npm run test
```
